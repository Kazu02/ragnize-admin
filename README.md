# RAGNIZE Admin Dashboard

Multi-tenant SaaS 管理プラットフォーム

## 機能

- **ダッシュボード** — KPI・MRR推移・製品別売上構成チャート
- **売上管理** — 売上明細・代理店別ランキング・CSV出力
- **請求管理** — 請求書発行・未入金管理・月次締め処理
- **代理店 / 孫代理店 / テナント / 契約 / 製品管理** — CRUD・検索フィルタ・CSV出力
- **課金単価設定** — 通話 / トークン / 月額の単価管理
- **通話利用管理 / トークン管理** — 利用ログ・チャート
- **レポート** — 各種分析チャート・CSVダウンロード
- **電子契約管理** — 送信・署名管理
- **お知らせ管理** — 公開・下書き管理
- **権限管理** — ロール別権限マトリクス
- **システム設定** — 会社情報・外部連携・通知設定・監査ログ

## 技術スタック

| ライブラリ | 用途 |
|-----------|------|
| React 18 | UI フレームワーク |
| Recharts | チャート描画 |
| Tailwind CSS | スタイリング |
| Babel Standalone | JSX トランスパイル（CDN） |

## 構成

単一HTMLファイル（`index.html`）で動作します。バックエンドAPI接続なしのフロントエンドプロトタイプです。

## ローカル確認

```bash
python -m http.server 8080
# http://localhost:8080 を開く
```

## AWS デプロイ予定

- S3 + CloudFront での静的ホスティング
- または EC2 / Amplify

## ロール切り替え

右上の「表示ロール」ドロップダウンから スーパーアドミン / 代理店 / 孫代理店 / テナント を切り替えられます。
